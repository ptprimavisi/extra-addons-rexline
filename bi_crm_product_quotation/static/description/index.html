CRM Product to Quotation Odoo App

Lead-Pipeline Product to Quotation Odoo Apps

Lead-Pipeline Product to Sales Order Odoo Apps

CRM Product to Quotation Odoo App helps users to easily create, manage, and track quotation directly from the pipeline view. This app integrates seamlessly with the CRM, allowing users to add multiple product to lead or pipeline, and create sales quotation directly from the CRM. This makes it easy to create accurate and detailed quotation that are tailored to the specific needs of each customer. In addition, the app allows users to track the quotation from CRM using smart buttons.

Features

Add Products to Lead/Pipeline

Users can add multiple products to crm lead/pipeline

Create Sales Quotation from CRM

User can create sales quotation with added product directly from crm.
Demo Request Video Tutorial

Add Products to CRM Pipeline

On CRM lead/pipeline view user can see "Products" tab, here user can add products as per customer need, and click "NEW DETAILS QUOTATION" button to create sales quotation.

Create Sales Quotation

On clicking "NEW DETAILS QUOTATION" button user can see new wizard will open with added products to lead/pipeline and click "SAVE" button.

Create Quotation from CRM

User can see "Quotations" smart button on lead/pipeline and view all created quotations from that.

User can also see all added products to crm lead/ppeline are added to sale quotations with same price and quantity.

Warning

If customer is not selected on crm lead/pipeline and click "NEW DETAILS QUOTATION" button then warning will raise.
